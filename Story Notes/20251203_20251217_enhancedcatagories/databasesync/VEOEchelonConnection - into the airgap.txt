VEOEchelonConnection - into the airgap
VEOEchelonConnection is the sql connection to the VEO Database for the particular stack. For the DEV environments that is the VEO_DEV data base on the dev-sql.veodesignstudio.com server and for the WBS stack in production that is the VEO database on the sql.veodesignstudio.com server. Other stacks can be found in the Library in Pipelines > Library.
This article is to identify in server code what we use from that database. Another facet of this is that we have synonyms into the veo database. This article will attempt to identify refrences for those synonyms.
